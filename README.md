# Vulnerability Assessment & Penetration Testing – Axia Africa

## Project Overview
This repository documents a **Vulnerability Assessment & Penetration Testing (VAPT) engagement** conducted on a web application called **bWAPP**. The engagement involved identifying and validating critical web application vulnerabilities and producing a professional security assessment report.

## Engagement Scope
- **Target:** bWAPP (intentionally vulnerable web application)
- **Vulnerabilities Tested:**
  - IDOR (Insecure Direct Object Reference)
  - SQL Injection
  - Cross-Site Scripting (XSS)
  - Command Injection
  - Broken Authentication

## Objectives
- Conduct comprehensive vulnerability testing.
- Validate exploits in a safe, controlled environment.
- Produce a detailed security assessment report with remediation recommendations.

## Deliverables
- `VAPT_Report.pdf` – Full engagement report with methodology, findings, and mitigation strategies.

## Skills Demonstrated
- Web application penetration testing
- Vulnerability identification and exploitation
- Security documentation and reporting
